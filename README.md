# Readme

## Quiz-Sheet
サイトURL: <br>
Googleスプレッドシートと連携してクイズを作成するアプリです

### 環境構築
DockerHub: https://hub.docker.com/r/ysk91/quiz-sheet<br>
```
$ docker pull ysk91/quiz-sheet
```
でDockerファイルをpull

```
$ docker-compose run -d
```
を実行し、 http://localhost:3000/ にアクセス
